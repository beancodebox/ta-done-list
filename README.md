# Ta-Done List

> "오늘 실제로 뭐 하면서 시간을 보냈는지" 기록하는 가장 빠른 방법

---

## 이 앱은

**"한 일"을 빠르게 기록하기 위한 초간단 도구입니다.**

투두 앱처럼 복잡하지 않습니다. 그냥:

- 입력창 하나
- 엔터 치면 저장
- 시간 자동으로 붙음
- 날짜별로 보임

---

## 왜 만들었나

기존 방식 (구글 킵):

1. 앱 열기
2. 날짜 찾기
3. 형식 맞추기
4. 입력

이 과정들이 **기록 흐름을 끊어서** 번거로웠습니다.

그래서 만들었습니다:

- **입력 마찰 최소화**
- 생각 안 하고 바로 기록
- 클릭 가능한 한 적게

---

## 사용 예시

```
2026-05-18

09:00 코드 리뷰
10:00 API 수정
11:00 점심
12:00 회의
```

하루하루 쌓이는 "회고용 작업 로그"입니다.

---

## 현재 기능

- ✅ 시간 자동으로 입력
- ✅ 엔터로 빠르게 기록
- ✅ 날짜별 조회
- ✅ localStorage에 저장 (브라우저 데이터 유지)

---

## 나중에 할 것들

지금은 집중하지 않는 것들:

- 로그인 / 회원가입
- 팀 기능, 공유
- 알림
- AI 기능
- 캘린더
- 태그, 복잡한 통계

기본이 견고해지고 나서 필요하면 추가.

## 개발

### 설치

```sh
pnpm install
```

### 개발 서버 실행

```sh
pnpm dev
```

### 빌드

```sh
pnpm build
```

### 린트

```sh
pnpm lint
```

---

## 기술 스택

- **Framework**: Vue 3
- **Bundler**: Vite
- **저장소**: localStorage (브라우저 로컬)
- **구조**: 단일 페이지 앱
